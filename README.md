## About

ML researcher working on explainability, multimodal AI, and agentic systems. This site includes my publications, news, blog articles, and CV.

Built with [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme, hosted on GitHub Pages.

---

## Running locally

Requires Ruby 3.3.x (use [rbenv](https://github.com/rbenv/rbenv)):

```bash
rbenv install 3.3.6
rbenv global 3.3.6
gem install bundler:2.5.7
bundle install
bundle exec jekyll serve
```

Open [http://localhost:4000](http://localhost:4000).
